# tabby-mobax-suite

[English](./README.md) | 한국어

**MobaX Suite** — 터미널에 항상 표시되는 **MobaXterm 스타일 왼쪽 사이드바**와 하단 **서버 상태 표시줄**을 추가하는 [Tabby](https://tabby.sh) 플러그인입니다.

사이드바는 항상 마운트되는 아이콘 레일로, 세 개의 탭과 tmux 액션으로 구성됩니다:

- **Sessions** — 저장된 SSH 프로필(그룹별); 클릭하면 연결됩니다. 프로필별 실시간 연결 점을 표시하고, 폴더(그룹)마다 커스텀 아이콘(Font Awesome 프리셋 또는 이미지 파일)을 지정할 수 있습니다.
- **SFTP** — 활성 SSH 탭에 연결된 자체 제작 파일 브라우저로, 셸의 작업 디렉터리를 **실시간으로 따라갑니다**(고정 토글로 멈춤). 다중 선택, 드래그 이동, OS 드래그-아웃/드롭 업로드, 잘라내기·붙여넣기 이동, 인라인 이름 변경, 키보드 탐색, 그리고 **로컬 편집** — 파일을 더블클릭하면 기본 텍스트 에디터로 열리고, 저장하면 자동으로 서버에 재업로드됩니다(MobaXterm 방식).
- **Macros** — 저장된 키 입력/명령 매크로이자 전역 **팔레트**(`Ctrl-Space`). 전역 또는 프로필별로 지정하고, 재사용 **변수**(`${name}`)와 JSON 가져오기/내보내기를 지원합니다.
- **tmux** (레일 액션) — 활성 SSH 탭에서 tmux 컨트롤 모드로 진입합니다. `tabby-tmux` 플러그인이 설치된 경우에만 표시됩니다.

레일 바깥에는 **하단 상태 표시줄**(활성 호스트의 실시간 CPU / 메모리 / 디스크 / 업타임 / 세션 + OS 로고)과 Tabby 설정 안의 통합 **설정 탭**이 있습니다.

## 호환성

- npm의 `tabby-*` 패키지 **`1.0.231-nightly.0`**(Angular `15.2.6`, rxjs `7.5.7`)에 맞춰 빌드됩니다. Tabby는 플러그인 API를 npm에 nightly 빌드로만 배포하므로 이 번호는 데스크톱 앱 번호보다 뒤처집니다 — 앱 `1.0.234`에서 정상 동작합니다. 이 고정 버전은 빌드 시점에만 쓰이고, 런타임에는 호스트 앱이 실제 인스턴스를 제공합니다. 사용 중인 Tabby가 크게 다르고 로드되지 않으면 해당 버전에 맞춰 다시 빌드하세요(`pnpm build`).
- **실시간 cwd 따라가기**는 셸이 OSC 1337 `CurrentDir`로 디렉터리를 보고해야 합니다(아래 zsh 예시). 안 되면 SFTP로 `/proc/<pid>/cwd`를 읽는 방식으로 폴백합니다(Linux 전용).
  ```sh
  precmd () { echo -n "\x1b]1337;CurrentDir=$(pwd)\x07" }
  ```

## 보안 참고사항

- **Vault 자동 잠금 해제 (선택).** vault 마스터 암호를 저장(OS 키 저장소로 암호화 — Windows DPAPI / Electron `safeStorage`, 평문 미저장)해 시작 시 자동으로 잠금 해제합니다. 이 기능은 마스터 암호의 보호를 **약화**시킵니다 — 잠금 해제된 OS 세션에 접근하는 누구나 저장된 SSH 비밀을 암호 없이 읽을 수 있으므로 **공유 PC에서는 켜지 마세요.** 암호를 지우면 비활성화됩니다.
- **로컬 드래그-아웃 헬퍼.** "파일을 바탕화면으로 드래그"는 로컬 전용 HTTP 서버(`127.0.0.1`, 무작위 포트)로 동작하며, 각 파일을 60초 후 만료되는 일회용 토큰으로 제공합니다.

## 설치

배포 후 Tabby 플러그인 관리자에서 설치(Settings → Plugins → `tabby-mobax-suite` 검색)한 뒤 **Tabby를 재시작**하면 적용됩니다.

## 소스에서 빌드

```bash
pnpm install
pnpm build      # -> dist/index.js (webpack production)
pnpm watch      # 변경 시 재빌드
pnpm test       # 순수 로직 단위 테스트 (vitest)
pnpm lint       # oxlint && eslint
```

### 개발 설치 (로컬 빌드 링크)

저장소를 Tabby 사용자 플러그인 디렉터리에 링크한 뒤 **Tabby를 재시작**하세요:

```powershell
New-Item -ItemType Junction -Path "$env:APPDATA\tabby\plugins\node_modules\tabby-mobax-suite" -Target "<이-저장소-경로>"
```

## 사용법

- 레일 아이콘을 클릭해 탭을 전환합니다. **활성** 아이콘을 클릭하면 창이 접히고 다시 클릭하면 펼쳐집니다. 너비·접힘 상태는 유지됩니다.
- **Sessions:** 프로필을 클릭해 연결하고, 그룹은 접기/펼치기됩니다. 폴더 우클릭 → **아이콘 변경...**으로 Font Awesome 프리셋이나 이미지 파일(png/jpg/gif/webp/svg)을 폴더 아이콘으로 지정합니다 — 이미지는 설정에 내장 저장되므로 원본 파일을 지워도 유지되고, 기본 아이콘으로 복원할 수 있습니다.
- **SFTP:** 활성 세션을 탐색하며 `cd`를 따라갑니다(**터미널 폴더 따라가기** 토글로 멈춤). 다중 선택(Ctrl/Shift+클릭 또는 마퀴), 드래그-드롭 또는 `Ctrl+X`/`Ctrl+V`로 이동, `F2` 이름 변경, `Delete` 삭제. 파일을 밖으로 드래그하면 내보내고, OS 파일을 드롭하면 업로드합니다. 파일을 더블클릭하면 설정된 텍스트 에디터로 열리며(기본: 메모장), 저장할 때마다 자동으로 재업로드되고 파일 권한도 유지됩니다. 이미지·압축 파일 등 알려진 바이너리 확장자는 OS 기본 프로그램으로 열리고, 컨텍스트 메뉴의 "연결 프로그램으로 열기"로 강제할 수도 있습니다.
- **Macros:** 만들기/편집/순서 변경, 또는 `Ctrl-Space`로 팔레트. 프로필 또는 모든 세션 대상 지정, 변수 모드에서 변수를 정의해 명령 단계에서 `${name}`으로 사용, JSON 가져오기/내보내기.
- **Settings:** Settings → MobaX Suite — 레일 항목(+ 선택적 Tabby 리로드 버튼, 기본 꺼짐), 상태 표시줄 + 폴링 주기, SFTP 기본 에디터(메모장 / Notepad++ / VS Code 프리셋 또는 직접 지정), vault 자동 잠금 해제를 설정합니다.
